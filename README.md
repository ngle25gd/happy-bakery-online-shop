# Happy Bakery Project
This is a bakery website for my assignment
## Features
- **Home Page**: Welcoming interface for users.
- **Product Catalog**: Displays 8+ bakery items using **CSS Grid**.
- **Product Detail**: In-depth information about specific treats.
- **Shopping Cart**: Fully functional cart with **LocalStorage** persistence.

## Tech Stack
- HTML5, CSS3 (Grid & Flexbox), JavaScript (Vanilla).

## Responsiveness
- Optimized for Desktop, Tablet, and Mobile views with multiple breakpoints.
